# 🛒 Amazon Clone

A responsive Amazon homepage clone built using **HTML5** and **CSS3**. This project recreates the user interface of Amazon's homepage, including the navigation bar, search bar, hero section, product cards, and footer. It was developed as a frontend practice project to strengthen HTML and CSS skills.

## 🚀 Live Demo

🔗 https://shopping-ui-clone.vercel.app/

## 📌 Features

- Responsive design for desktop, tablet, and mobile devices
- Amazon-inspired navigation bar
- Search bar with category selection
- Hero banner section
- Product category cards
- Hover effects
- Multi-section footer
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome

## 💻 Getting Started

### Clone the repository

```bash
git clone https://github.com/Nishthaparve/Amazon-Clone.git
```

### Navigate to the project folder

```bash
cd Amazon-Clone
```

### Run the project

Open the `index.html` file in your browser or use the **Live Server** extension in Visual Studio Code.
